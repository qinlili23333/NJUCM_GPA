# NJUCM_GPA
南中医绩点计算器  

# 规则  
南中医绩点计算规则为除体育外所有必修课的学分X（成绩-50）X0.1之和除以学分之和  

# ALL.js  
用于计算有史以来所有课的绩点  
请登录教务系统后打开这个页面 http://202.195.210.218/gradeLnAllAction.do?type=ln&oper=sxinfo&lnsxdm=001 使用  

# SINGLE.js  
用于计算最近一个学期的绩点  
请登陆教务系统后打开这个页面 http://202.195.210.218/bxqcjcxAction.do 使用
